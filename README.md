# Stock Market Dashboard Analysis

## Project Overview
This project focuses on analyzing stock market data to understand portfolio performance,
trading activity, and market trends. Interactive dashboards are created to convert
complex financial data into clear and actionable insights for decision-making.

## Business Objective
- Monitor overall portfolio performance
- Analyze returns, volatility, and dividend yield
- Identify top-performing stocks and sectors
- Support data-driven investment decisions

## Tools Used
- Microsoft Excel (data cleaning and preliminary analysis)
- MySQL (data querying and transformation)
- Microsoft Power BI (interactive dashboard creation)
- Tableau (data visualization and dashboard design)

## Key Metrics (KPIs)
- Portfolio Value
- Portfolio Return Percentage
- Trader Profit and Loss
- Dividend Yield
- Volatility
- Average Daily Trading Volume

## Key Insights
- Portfolio value analysis by ticker to identify high-contributing stocks
- Sector-wise market capitalization to understand sector dominance
- Dividend yield comparison across exchanges such as NSE and NASDAQ
- Identification of high-volume trades using Top 10 Trade IDs

## Dashboard Features
- Interactive filters for ticker and sector selection
- Dynamic KPIs that update based on user selection
- Visual comparison across exchanges, sectors, and geographies
- User-friendly design for quick analysis

## Files in This Repository
- **Data/** – Raw and processed datasets
- **Stock Market Excel Dashboard.xlsx** – Excel-based analysis
- **Stock Market.pbix** – Power BI dashboard file
- **Stock Market Dashboard.twbx** – Tableau dashboard file
- **Queries.sql** – SQL queries used for data analysis
- **images.jpg** – Dashboard screenshots and visuals
